 \mbox{\hyperlink{class_addressbook}{Addressbook}} viewer is G\+UI a C++ program using Qt framework for drawing and straightforward manipulation that join predefined corresponding rows between two Sqlite database tables. Based on the model/view architecture. Read detailed \href{task/test.pdf}{\texttt{ description}} of a program.
